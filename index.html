Here you can find descriptions and samples of some of the PUPs (Potentially Unwanted Programs) that I've found floating around George Mason High School.

  • Spotify Premium(12 Month)

    This PUP is a simple AppleScript which on run will repeatedly open Finder windows, speak the word "wat," and turn up the volume to 100%. The process is called "memes." You can kill it quite easily with kill $(ps aux | grep '[S]potify Premium' | awk '{print $2}'). Sometimes it will be a login item, in which case you'll need to remove it from the user's list. You can press Command + Option + W in order to kill all Finder windows.

    The AppleScript code is as follows:

    repeat 9999 times
	tell application "Finder" to make new Finder window
	set volume output volume 100
	say "wat" using "Agnes" without stopping current speech
end repeat
